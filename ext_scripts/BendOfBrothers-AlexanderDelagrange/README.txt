BEND OF BROTHERS MS

install notes:

Unzip all the files from the Rar into any folder on your computer.
Open max
MAXScript-> OpenScript->Open: BendOfBrothersMacro.ms
Inside MAXScript editor with BendOfBrothersMacro open-> Tools-> Evaluate All
Inside 3dsMax->Customize->Customize User Interface...
Toolbars tab->
If you dont have your own toolbar yet Hit the New Button and name it whatever you want->Hit OK-> drag the new window and dock it anywhere you want in your max UI
While still in the Toolbar tap check Category: AlexanderDelagrangeScripts-> Select Bend of Brothers text under Action and drag it in your new toolbar.
Done
